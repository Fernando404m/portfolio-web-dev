Habilidades

HTML5

O HTML (HyperText Markup Language) é a linguagem de marcação mais usada na criação de sites, devido à sua semântica poderosa que auxilia o sistema de busca do Google.
Estou atualmente estudando mais sobre sua semântica, para facilitar o design de minhas aplicações web.

CSS

O CSS (Cascading Style Sheets) é o que define os visuais do HTML, em outras palavras, é o componente essencial para a criação de designs dos elementos HTML.
Eu sempre busco manter um site organizado, responsivo e adaptável ao público.

JavaScript

JavaScript é uma linguagem de programação que define funções a serem executadas em um site. Juntamente com o CSS e HTML, o JavaScript é uma das três principais tecnologias da web (World Wide Web).
Eu já estou bem acostumado com a lógica de programação e creio estar me dando bem com o JavaScript e sua sintaxe.
